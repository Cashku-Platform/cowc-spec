# Changelog

All notable changes to the COWC specification will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-03

### Added
- Initial COWC specification release
- OAuth 2.0 authentication (Client Credentials flow)
- User linking and consent management
- Fund Discovery API
- Portfolio API
- Order API (preview, create, status)
- Embedded Flows specification
- Webhook specification
- Security requirements
- Extensibility guidelines
- Sample code: Node.js, Python, Flutter, Swift, Kotlin
- Postman collection
- Cashku provider implementation (Appendix A)

### Providers
- Cashku (Malaysia) - Initial provider
